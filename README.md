## Executar Backend

Acessar a pasta do backend </br>
``cd ./backend``

Instalação de dependências </br>
``npm i``

Executar o projeto </br>
``nodemon ./server.js``
<br/>
<br/>


## Executar Frontend

Acessar a pasta do frontend </br>
``cd ./frontend``

Instalação de dependências </br>
``npm i``

Executar o projeto </br>
``npm start``
